# Speedrunify Software

### this software will help you in running speedrun and focus mode to see the time

> ##### GUI built in Raylib

##
#### Features:
- 2 Mode in 1, Stopwatch for speedrun and Clock for focused mode.
- Adjustable time format, 24 Hour or Am/Pm.
- Pinned toggle, for set this app running on top of another application window.
##

> [!WARNING]
> **THIS SOFTWARE IS STILL UNDER DEVELOPMENT, Windows 10 or later only supported, the beta V1 version that will be released.**

##

> #### Note: 
> #### **if you want to developed/build yourself or even debug, you'll need these:**
> ##### - MSTS2 MINGW64 for the installation header
> ##### - GCC, GNU Compiler
> ##### - Linker (C-MAKE, etc.)
